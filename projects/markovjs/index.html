Markov Chain Text Generator

This is a Markov chain text generator that will "learn" the style of a given input text and can randomly generate text to imitate that style. You can enter any body of text, but I recommend something relatively substantial (at least a few thousand words). A long news article, Wikipedia page, or Project Gutenberg text will work well.

You can then choose the basic element of each chain: a letter or a word. The benefit of choosing a word is that you'll only receive valid words as output. However, I would then recommend using a shorter chain length (not more than 2 or 3) otherwise you'll see lines lifted wholesale from the source text. Parsing on the letter is sometimes more interesting in that there's larger deviation from the source while the style remains intact.
